# Memory Bank

This directory contains the project's memory and knowledge base for ChainEquity.

## Structure

- `activeContext.md` - Current active context and ongoing work
- `projectBrief.md` - Project overview and goals
- `requirements.md` - Detailed project requirements and specifications
- `systemPatterns.md` - System architecture and design patterns
- `techContext.md` - Technical context and decisions

## Purpose

The memory bank serves as a knowledge repository for the project, helping maintain context across development sessions and documenting important decisions, patterns, and learnings.
